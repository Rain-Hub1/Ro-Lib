# `Ro` Lib

## `Demo`
```lua
local Lib = loadstring(game:HttpGet("https://raw.githubusercontent.com/Rain-Hub1/Ro-Lib/refs/heads/main/Source.lua"))()

Lib:Demo()
```

## `Load`
```lua
local Lib = loadstring(game:HttpGet("https://raw.githubusercontent.com/Rain-Hub1/Ro-Lib/refs/heads/main/Source.lua"))()
```

## `Window`
```lua
local Win = Lib:Window({
  Title = "My Window!"
})
```

## `Tab`
```lua
local Tab = Win:Tab({
  Name = "My Tab!"
})
```
